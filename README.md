# Automation Testing with CI/CD

This project demonstrates automation testing integrated with a CI/CD pipeline.

## Description
- Automated tests executed automatically via CI/CD
- Test results generated after each run
- Basic continuous integration setup

## Technologies
- Selenium WebDriver
- C#
- NUnit
- GitHub Actions (CI/CD)

## Purpose
- Learn CI/CD concepts in automation testing
- Practice running tests automatically in pipelines
